Sale Order Line Views (List View, Form View, Kanban View, Search View)

This module useful to show sale order lines products and other information related to it.

How it works? Click here to watch video

Key features:
  • Easy to filter quotation lines today, yesterday, current week, previous week, current month, previous month, current year, previous year.
  • Easy to group by quotation lines customer, sales order, product.
  • Easy to filter sale order lines today, yesterday, current week, previous week, current month, previous month, current year, previous year.
  • Easy to group by sale order line customer, sales order, product.
  • Very useful to track records of sale order lines of customers who have many sale orders.
  • Easy to get group by products of sale order lines.

Quotation lines contains all order lines of all order that are in 'draft' stage.

Filter records in quotation lines list view on "Order Date" that are ordered in previous year. Can filter record using any of provided custom date filters list.

Group by quotation lines list view records by "Customer". Can also group by "Product" and "Order".

Kanban view of quotation lines group by "Quotation".

Order lines contains all sale order lines of all order that are in 'sale' or 'done' stage.

Filter records in order lines list view on "Order Date" that are ordered in previous week. Can filter record using any of provided custom date filters list.

Filter record in order lines list view records by "Customer". Can also group by "Product" and "Order".

Kanban view of order lines group by "Order".

Filter record in Kanban view of order lines by "Previous Month".

Can "Edit" record of "Quotation Line" or "Order Line", this menu link visible only to user with group "Sales Manager".
